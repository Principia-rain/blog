# 使用Github搭建自己的博客

前言：本博客就是使用Github上的开源项目构建的。接下来，本文将从零开始给出搭建博客的详细流程。

## 1. 注册github账号

这是最初的一步，没有账号接下来的步骤都无法执行。

github链接：[github](https://github.com)

## 2. 打开skunk-html并添加至自己的仓库

注册完账号后打开 [skunk-html](https://github.com/mg0x7BE/skunk-html) ，进入项目后点击 **Fork** 将其加入至自己的仓库

## 3. 设置仓库

请看 [设置](https://mg0x7be.github.io/skunk-html/setup-guide.html) 。一步步跟随其操作即可。

## 4. 上传博客文章

用 **Markdown** 语言在本地写博客，将 .md 格式的文件上传到你自己仓库的 **markdown-blog** 文件夹中。将文件名设为日期，如 2000-1-1。文章内开头的大标题会被设定为该片博客的名称。

关于如何使用Markdown，[基本引导]([SkunkHTML - Setup guide](https://mg0x7be.github.io/skunk-html/setup-guide.html)) 。

## 5. 博客上线

刷新页面，等待一会博客就上线了！



更详细的指引可以看官方文档：[详细指南](https://github.com/mg0x7BE/skunk-html/blob/master/README.md) 。
